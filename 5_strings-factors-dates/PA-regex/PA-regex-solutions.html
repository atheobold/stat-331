Decode a message

Complete the following steps to decode the message.

  1. Remove any spaces before or after each word.
message <- str_trim(message, side = "both")
  1. No word should be longer than 16 characters. Drop all extra characters off the end of each word.
message <- message %>% 
  str_sub(start = 1, end = 16)
  1. Any time you see the word âughâ, with any number of hâs, followed by a punctuation mark, delete this.
message <- message %>% 
  str_replace_all(pattern = "ugh+[!,?.;]+", 
                  replacement = "")
  1. Replace all instances of exactly 2 aâs with exactly 2 eâs.
message <- message %>% 
  str_replace_all(pattern = "[aa]{2}", 
                  replacement = "ee")
  1. Replace all zâs with tâs.
message <- message %>% 
  str_replace_all(pattern = "z", replacement = "t")
  1. Every word that ends in b, change that to a y.
message <- message %>% 
  str_replace(pattern = "b$", replacement = "y")
  1. Every word that starts with k, change that to a v.
message <- message %>% 
  str_replace(pattern = "^[kK]", replacement = "v")
