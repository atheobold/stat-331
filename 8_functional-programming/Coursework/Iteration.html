✅ Check-in: Functional Programming

Question 1:

Suppose we would like to find the median of the measurement variables (bill_length_mm, bill_depth_mm, flipper_length_mm, body_mass_g) from the penguins dataset (from the palmerpenguins package). Which of the following will produce a vector of the median of every measurement variable?

Select all that apply!

(a)

penguins %>% 
  select(bill_length_mm:body_mass_g) %>% 
  map(median, na.rm = TRUE)

(b)

penguins %>% 
  select(bill_length_mm:body_mass_g) %>% 
  map_dbl(median, na.rm = TRUE)

(c)

penguins %>% 
  select(bill_length_mm:body_mass_g) %>% 
  sapply(., median, na.rm = TRUE)

(d)

penguins %>% 
  select(bill_length_mm:body_mass_g) %>% 
  lapply(., function(x) median(x, na.rm = TRUE))



Recall that in the last unit, we discussed the challenge of standardizing many columns in a data frame. For example, we wanted to standardize all the numeric variables in the penguins dataset.

Here is the function we wrote for that:

standardize <- function(data) {
  deviations <- data - mean(data, na.rm = TRUE)
  newdata <- deviations / sd(data, na.rm = TRUE)
  return(newdata)
}

Question 2: Why does the following not work?

penguins %>%
  mutate(
    body_mass_g = map_dbl(body_mass_g, standardize)
  )
